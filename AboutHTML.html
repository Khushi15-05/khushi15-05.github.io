What is HTML?

  • HTML is the standard markup language for creating Web pages.
  • HTML stands for Hyper Text Markup Language.
  • HTML describes the structure of a Web page.

HTML Documents

  • All HTML documents must start with a document type declaration: <!DOCTYPE html>.
  • The HTML document itself begins with <html> and ends with </html>.

HTML Elements

  • An HTML element usually consists of a start tag and an end tag, with the content inserted in between:
    <tagname>Content goes here...</tagname>

HTML Attributes

  • Attributes provide additional information about an element.
  • Attributes usually come in name/value pairs like: name="value".

Basic HTML tags

  • <body> Defines the visible part of the HTML document.
  • <head> Defines information about the document.
  • <title> Defines a title for the document.
  • HTML paragraphs are defined with <p> tag.
  • Headings are defined with the <h1> to <h6> tags.
    <h1> defines the most important heading and <h6> defines the least important heading.

Writing the basic program



<!DOCTYPE html>
<html>
<head>
     <title>Webpage</title>
</head>
<body>
     <h1>My First Heading</h1>
     <p>My first paragraph.</p>
</body>
</html>



Some other tags

<img> Defines an image
<a> Defines the hyperlinks
<button> Defines a clickable button
<marquee> To create moving text or image
<ul> To create unordered list
<ol> To create ordered list
<li> Defines list items marked with either disc, circle, square, a, A, i, I, 1 or none.
<table> To create a table
<tr> To create a row in the table
<th> Defines table heading (text appears in bold)
<td> Defines table data
<thead> Text written between this tag always appears in the top row of the table.
<tfoot> Text written between this tag always appears in the bootom row of the table.
<tbody> Text written between this tag appears between the top and bottom row.
<b> Defines bold text
<strong> Defines important text
<i> Defines italic text
<em> Defines emphasized text
<u> Defines underlined text
<ins> Defines inserted text
<del> Defines deleted text
<mark> Highlights the text
<small> Decreases the size of the text
<sub> To place a text in subscript
<sup> To place a text in superscript
<abbr> Defines an abbreviatin or acronym
<address> Defines the contact information of the author of a document in italics
<cite> Defines the title of a work or citation in italics
<bdo> bdo stands for Bi-Directional Override and is used to override the current text direction
<map> Used to define a client-side image-map. An image-map is an image with clickable areas.
<area> Defines area inside an image map
<center> Aligns the text to center
<code> Defines a piece of computer code
<pre> Defines pre-formatted text
<hr> Defines a thematic change in the content by giving a horizontal line
<br> Gives a line break
<div> Defines a section in a document
<iframe> Defines an inline frame
<picture> Defines a container for multiple image resources
<source> Defines multiple media resources for media elements
<span> Provides a way to add a hook to a part of a text or a part of a document
<style> Defines style information for a document
